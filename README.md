# stns-passwd
This command is used to generate a password to be used in STNS

## Description

## Usage
```bash
$ stns-passwd <password_strings>
```

## options
```bash
Usage of stns-passwd:
  -version
        Print version information and quit.
```

% brew tap stns/stns_passwd
